Getting Started

Congratulations on installing and executing the Kepler Recommendation Engine.
  • Upload one or more data files.
  • Define a configuration for your recommendation system in a JSON file.
  • Upload the configuration file to the recommendation system.
  • Parse the configuration file for issues and edit to make corrections.
  • Process the data files.
  • Make recommendations.
